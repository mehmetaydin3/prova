Marketplace for musicians to showcase services and get paid work globally.

## The global marketplace for musicians

Prova helps musicians find paid work and helps clients discover professional musicians worldwide.

### Services musicians can offer
- Live performances
- Wedding gigs
- Recording sessions
- Remote session work
- Music lessons
- Composition & arranging
